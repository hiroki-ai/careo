# Careo キャリアセンタープラン 価格表

最終更新日：2026年03月23日

## プラン構成

| プラン | 想定規模 | 年額（税抜） | 含む機能 |
|--------|----------|-------------|---------|
| **Starter** | ~500名 | ¥480,000 | Phase 1のみ |
| **Standard** | ~1,500名 | ¥960,000 | Phase 1 + Phase 2 |
| **Enterprise** | 3,000名〜 | 要見積もり | Phase 1〜3 + SLA |

## 各フェーズの機能範囲

**Phase 1 (MVP)**
- 学生一覧・基本プロフィール閲覧
- OB/OG訪問承認ワークフロー
- 全体アナウンス送信
- 基本ダッシュボード

**Phase 2**
- 面談記録
- セグメント配信（学年/学部/状況別）
- 筆記試験進捗ビュー
- イベント管理

**Phase 3**
- 高度な分析レポート
- PDF報告書出力（文科省・就職委員会向けフォーマット）
- スタッフ権限ロール細分化
- アクセスログ監査

## 上智大学（特別条件）

運営資金（現物支援 or 資金提供）+ 公認肩書き + キャリアセンター内宣伝 と交換。
金額・条件は別途交渉。

## 補足条件

- 初年度トライアル割引：20%OFF（導入ハードルを下げる）
- 複数年契約：5%割引
- 導入支援（初期設定・スタッフ研修）：¥100,000/回（オプション）
